# Rail Redesign — Implementation Report

## Status: COMPLETE
All 13 reference scenarios (A–M) pass. All 237 lib tests pass. `#[ignore]` removed.

## Changes made

### src/render.rs

1. **Strip trailing `\n` from render_rail output** — The `grid()` test helper sets height = `ansi.lines().count()`. A trailing `\n` caused vt100 to scroll away the first line (RADAR header). Fix: `out.pop()` at end of `render_rail()` before returning.

2. **Remove right-slot** — `right_slot()` always returns `String::new()`. No more elapsed, done/total, "done", "failed" in the tab header line.

3. **Remove leading space on inactive tab rows** — `bar` for inactive tabs changed from `" ".to_string()` to `String::new()`. `bare_min` made conditional on `row.active` (0 or 1 bar width). Glyphs now sit at col 0 for inactive tabs.

4. **Remove braille spinner from detail message** — `msg_spin` no longer appended to detail activity. Status glyph still spins.

5. **Single-pane Done shows detail line** — `row_lines` and `render_row` detail-emit now treat `Done` like `Running/Error/Pending` (shows detail when msg non-empty).

6. **`is_multi_pane` counts only tracked panes** — Changed from `display.panes.len() > 1` to counting `p.is_tracked()` panes > 1. Untracked panes (shell prompts) no longer make a tab multi-pane.

7. **Line-per-pane multi-pane rendering** — Replaced `pane_tree_plan` / collapse tree with new `emit_pane_line` function. Each tracked pane gets one line: `  {glyph} {mark} {msg}` (inactive) or `▌ {glyph} {mark} {msg}` (active). `pane_tree_plan` and `PaneTreePlan` retained for existing unit tests.

8. **Safety cap of 6 pane lines** — `const MAX_PANE_LINES: usize = 6`. Beyond cap, emits `  +N more` line (or `▌ +N more` when active).

9. **`row_lines` for multi-pane** — New formula: `1 + min(tracked_count, 6) + (1 if tracked_count > 6 else 0)`. No longer depends on `active`.

10. **Overflow strip format** — Changed from `○ ○ ○ ── +N idle ▾` to `+N idle ▾`.

11. **Header overflow format** — `N▲` (no space) instead of `N ▲`.

12. **Removed `emit_child_line`** — Dead code after multi-pane rewrite.
